Workshop 8: Forms and JavaScript

Workshop 8

In this workshop, you’ll learn how to create a simple form that collects data from the user, and uses JavaScript to process and display a result. We’ll build an interactive age calculator.

Age Calculator

We'll create a form where the user can input their birth year, and JavaScript will calculate their current age based on the current year.

Example Form:


<form>
    <label for="birthYearInput">Enter your birth year:</label>
    <input type="number" id="birthYearInput" placeholder="Enter your birth year">
    <button type="button" onclick="calculateAge()">Calculate Age</button>
</form>
<p id="ageResult"></p>

JavaScript Code:


function calculateAge() {
    let birthYear = document.getElementById("birthYearInput").value;
    let currentYear = new Date().getFullYear();
    let age = currentYear - birthYear;

    if (birthYear === "" || birthYear > currentYear) {
        document.getElementById("ageResult").innerText = "Please enter a valid year.";
    } else {
        document.getElementById("ageResult").innerText = "You are " + age + " years old.";
    }
}

Text Reverser

Now let’s create a fun tool that takes a string of text from the user, reverses it, and displays the reversed text back to the user.

Example Form:


<form>
    <label for="textInput">Enter text to reverse:</label>
    <input type="text" id="textInput" placeholder="Enter your text">
    <button type="button" onclick="reverseText()">Reverse Text</button>
</form>
<p id="reversedText"></p>

JavaScript Code:


function reverseText() {
    let text = document.getElementById("textInput").value;
    let reversed = text.split("").reverse().join("");

    if (text === "") {
        document.getElementById("reversedText").innerText = "Please enter some text.";
    } else {
        document.getElementById("reversedText").innerText = "Reversed text: " + reversed;
    }
}
